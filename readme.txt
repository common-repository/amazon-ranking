== Description ==

This widget shows Amazon Bestsellers, Hot New Releases, Most Gifted and Most Wished For.

You can

* choose a shop among Amazon.com, Amazon.co.uk, Amazon.de, Amazon.fr, Amazon.co.jp and Amazon.ca.
* register node id to limit ranking to specific category. (e.g. Node id for books on programming is "3839" at Amazon.com.)
* register your associate ID to earn money.
* place more than one ranking widget in your page.
* include hit item names in your page automatically.

== Installation ==

1. Extract all the files
1. Upload the folder `amazon-ranking` to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. Activate the widget through the `Widgets` menu in WordPress
